# Rock Paper Scissors Game

An interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**.
The user selects Rock, Paper, or Scissors, and the computer generates a random move to determine the winner.

## Features

* User vs Computer gameplay
* Random computer move generation
* Win / Lose / Draw result detection
* Dynamic score tracking
* Interactive UI with visual feedback

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How to Play

1. Click on **Rock**, **Paper**, or **Scissors**.
2. The computer randomly selects its move.
3. The game decides the winner based on standard rules:

   * Rock beats Scissors
   * Scissors beats Paper
   * Paper beats Rock
4. The score updates automatically after each round.

## Project Structure

* `index.html` – Game layout
* `style.css` – Styling and design
* `app.js` – Game logic and functionality
